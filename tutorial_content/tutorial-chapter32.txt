        <CodeBlock filename="src/web/server.ts" lang="typescript">{`// Web 版——通过 WebSocket 把 arun() 连接到浏览器前端
import { createServer } from "node:http";
import { WebSocketServer } from "ws";
import { arun } from "../harness/agent.js";
// ...

const wss = new WebSocketServer({ port: 8080 });

wss.on("connection", (ws) => {
  let transcript = new Transcript();
  const registry = createRegistry();

  ws.on("message", async (raw) => {
    const { type, payload } = JSON.parse(raw.toString());

    if (type === "user_message") {
      // 流式事件通过 WebSocket 发送给前端
      const events: StreamEvent[] = [];

      transcript = await arun(
        provider,
        registry,
        payload,
        transcript,
        systemPrompt,
        (event) => {
          ws.send(JSON.stringify(event));
          events.push(event);
        },
      );
    }
  });
});`}</CodeBlock>

        <p>Web 版的核心挑战是<strong>权限模型</strong>：在终端中，确认对话框弹在用户面前；在 Web 中，用户可能不在浏览器前。Web 版需要：① 异步确认通知（消息推送到前端等待回复）；② 超时自动拒绝；③ 可选的"信任模式"（对特定操作不要求确认）。</p>
